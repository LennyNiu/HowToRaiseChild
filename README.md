# HowToRaiseChild

0 ~ 18岁育儿手册及常见问题解答。

日本作家伊坂幸太郎曾在其作品《一首小夜曲》中说过一句话：一想到为人父母居然不用经过考试，就觉得太可怕了。
> 親になるのに資格試験がない、というのが恐ろしい 【アイネクライネナハトムジーク】

所以我创立本仓库，希望更多的人抛弃“只要有了孩子就可以成为合格父母”的心态。本着科学、理性的态度来准备、梳理、学习养娃过程中遇到的各种问题，并以此来实现父母的自我成长和养育能力的更新迭代，实现对子女的良好教育。

**特别说明**

本仓库主要目标是收集现实操作中父母的经验和案例反馈（而不是大而全的“百科”），构建养娃经验手册方便大家检索参考，未来计划添加针对各个年龄段父母考题和子女反馈模块。

如果需要完整的育儿百科，推荐日本作家松田道雄的《[育儿百科](https://book.douban.com/subject/1101921/)》，该书出版已半个世纪，一直在不断更新，2010版中文译本超过80万字，涵盖了从孕期到宝宝幼儿园的几乎所有日常内容。建议新手父母人手一本。


欢迎广大Follower交流、总结、提交PR。

# 关于作者

1986年生人，男，现居上海，2008年大学毕业，本职工作是互联网技术岗，在2020年和2023年各育有一子。

悲观主义，行动派；自幼喜欢阅读写作，过去二十年阅读超过600本书，写作超过200万字；热衷于持续学习和迭代，相信学习和知识的力量。

自2010年起，工作之余长期参与心理支持、青少年教育和运动培训相关的公益活动，具体如下：

- 300+小时中小学生补课经验，主要科目：作文、数学、物理、英语、口语、阅读推广
- 500+小时公益心理支持经验，涉及领域：亲密关系、家庭关系、青少年心理
- 1500+小时体育运动培训经验，涉及运动：羽毛球、足球、网球、飞盘

作者育娃理念：

- 我和爱人日常都是极简主义，对于养娃也秉承“极简”的大方向，推车、衣物鞋袜、玩具、书籍登靠亲戚朋友送的都已足够。至今两个娃除了奶粉尿布，其余日常物品花的钱超不过1000块。
- 坚持自己带娃，优先请住家阿姨，而不是让爷爷奶奶或外公外婆参与教育。
- 重视和娃的陪伴、沟通互动和独立性。家里电视从来不开，手机也基本不让玩。大宝1岁半开始自己吃饭；鼓励户外运动，2岁不到晒得黝黑也在所不惜。
- 关注基础的语言表达和沟通能力、基本的手眼身体协调性，从小跟随我们训练，将足球、羽毛球、飞盘作为日常玩具之一。
- 下定决心不鸡娃，不论运动还是课程，尽可能自己教。报班只以娃自己的兴趣为主导，不盲目报班。
