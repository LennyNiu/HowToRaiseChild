# HowToRaiseChild
## 序
0 ~ 18岁育儿指南及常见问题解答。

据说日本作家伊坂幸太郎曾说过一句话：“一想到为人父母居然不用经过考试，就觉得真是太可怕了。” 

所以我希望抛弃那些天然就可以成为完美父母的心态，创立本仓库。本着科学、理性的态度来梳理养娃过程中遇到的各种问题，以此来实现父母的自我成长和养育能力的更新迭代，实现对子女的良好教育。

目前主要收集知识构建养娃手册，未来计划添加父母考题和为人子女反馈模块。

欢迎广大Follower学习、交流、总结、推送。
